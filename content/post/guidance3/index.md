# 大三上
## 小学期——计算机专业基础实习
- 你可知道 BIT101 上**评价最多**的课程？
- 你是否目睹过课程评价为 **0.81** 的课程？
- 你有没有在群聊中对**东软**有所耳闻？

恭喜你！欢迎来到北理计院特色，究极的试炼——**计算机专业基础实习**！

我本人在 BIT101 对应课程中也有匿名评论，因为太长这里就不放出来了。

以下是该课程的规则怪谈：
1. 每天一定要准点到班上扫码
2. 不能完成第一条，则需要由朋友想办法帮忙扫
3. 做小测前一定要打开AI，并将相关文档作为上下文输入
4. 不要盲从老师的环境配置与要求，往往Linux/Ubuntu以及Qt版本都可以更换
5. 不一定需要使用虚拟机提供Linux环境，使用WSL、双系统、远程服务器开启桌面等方式都可以
6. 在遵循第四条时，需要在汇报时假装使用了老师要求的版本
7. 在大教室中注意自己笔记本的电量，身体温度，以及耳朵所能承受的分贝上限
8. 一定要小组熟练使用 Codex / Claude Code 等 Coding Agent [^1][^2][^3]
9. 忽略老师对你的评价
10. 做好答辩时被提问的准备
11. 面对第八条可能产生的Bug，最后阶段可以直接避免展示不好的部分，利用好你的视频剪辑技术与演讲能力
12. 小组使用 Git 维护代码，并提前配置好统一的环境（遵循第四条）
13. 一时兴起时，记得在 BIT101 评论
14. 记住课程的名称不是《环境科学与技术》《压力管理与心理韧性》《Vibe Coding与AI Agent实践》《公共演讲与辩论》

课程总共两者，实际上从开始到答辩可能只有一周半不到的时间，加上还有中期答辩和各种文档。最后均分 86.55，看上去还行，但实在折磨。
