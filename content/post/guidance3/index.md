## 大三上

这学期许多课设计计算机系统，时间充裕可以看看相关科普书与视频，例如《计算机底层的秘密》[^6]、Core Dumped[^7]（该频道可以从头开始看）。

### 小学期——计算机专业基础实习
- 你可知道 BIT101 上**评价最多**的课程？
- 你是否目睹过课程评价为 **0.81** 的课程？
- 你有没有在群聊中对**东软**有所耳闻？

恭喜你！欢迎来到北理计院特色，究极的试炼——**计算机专业基础实习**！

我本人在 BIT101 对应课程中也有匿名评论，因为太长这里就不放出来了。

以下是该课程的规则怪谈：
1. 每天一定要准点到班上扫码
2. 不能完成第一条，则需要由朋友想办法帮忙扫
3. 做小测前一定要打开AI，并将相关文档作为上下文输入
4. 不要盲从老师的环境配置与要求，往往Linux/Ubuntu以及Qt版本都可以更换
5. 不一定需要使用虚拟机提供Linux环境，使用WSL、双系统、远程服务器开启桌面等方式都可以
6. 在遵循第四条时，需要在汇报时假装使用了老师要求的版本
7. 在大教室中注意自己笔记本的电量，身体温度，以及耳朵所能承受的分贝上限
8. 一定要小组熟练使用 Codex / Claude Code 等 Coding Agent [^1][^2][^3]
9. 忽略老师对你的评价
10. 做好答辩时被提问的准备
11. 面对第八条可能产生的Bug，最后阶段可以直接避免展示不好的部分，利用好你的视频剪辑技术与演讲能力
12. 小组使用 Git 维护代码，并提前配置好统一的环境（遵循第四条）
13. 一时兴起时，记得在 BIT101 评论
14. 记住课程的名称不是《环境科学与技术》《压力管理与心理韧性》《Vibe Coding与AI Agent实践》《公共演讲与辩论》

课程总共两周，实际上从开始到答辩可能只有一周半不到的时间，加上还有中期答辩和各种文档。**1学分，最后均分 86.55**，看上去还行，但实在折磨。

### 小学期——大数据系统开发

上述《计算机专业基础实习》占据了小学期的前两周，而《大数据系统开发》是后一周。考试时间会在之后，可以留几天时间复习。

这门课是公式的背ppt知识点，以及将往年题的答案背一背。实验方面AI都可以搞定。**1学分，均分84.45**。

### 计算机系统导论
这门课源自 CMU 的 CSAPP[^4]，知识面很广，认真学下来也会有收获。最终考试与往年题重合度很高。**课程有5学分，很重要。平均分80.45**。

这门课程入门可以看九曲阑干的讲解[^5]，然后利用AI看相应PPT与书本，尤其关注书本上的练习题。如果时间实在充裕，也可以考虑做一点北大的CSAPP试卷[^8]（较难）。

参考 BIT101 上的课程评论，往年卷结构与具体题目如下：
- 20道选择题，各种小的知识点，主要是第1章与第10-12章。
- 8道大题，覆盖2-9章，每章一个。

下面`电子书页码 = 课本页码 + 36`。

我按照上一级(22)与我这一级(23)，大概描述一下大题（从1开始编号，因此对应章节为编号加一）：
1. - (22)给定无符号数/有符号数补码，看二进制相加后是否溢出。
    - (22、23)IEEE 754浮点数表示方法，主要是32位，要熟练各个种类（规约形式、非规约形式、无穷与NaN），以及给定小数转换成相应IEEE浮点数表示，或者反过来。
    - (23)判断几个C语言相关的表达式是否正确，不为真时给出反例，参考练习题2.44、2.54。
2. - （22）给定函数的C语言与汇编语言，画出栈结构，分析缓冲溢出，改善代码。
    - （23）家庭作业3.64，给定C语言与汇编语言，考三维定长数组的地址计算，分析寄存器变化。
3. - (22、23)练习题4.13-4.14类型题目，也就是每个指令在各个阶段的操作。可以带着理解记忆。23级考了jxx从取值到新PC每阶段的表格，课本270页图4-21。
   - (22、23)课本318页，背图4-66，会直接挖空让你填写，好像占5分多。我这一年考的是这个表，不确定会不会考其他表，时间紧也可以就这么背不关心它们的区别。（事实上我都没背下来，在这里扣了几分）
4. - (22、23)练习题 5.5 与 5.6 类似题目，考运算次数、CPE，优化。23级加了一问写成2*1a循环。
   - (22)写出局部优化和全局优化的具体技术，以及影响优化的因素（内存别名和函数调用，课本344页）。
    - (22)课本370页，参考图5-21，写一个循环展开（例如2x2或者3x3）。
5. - (22、23)高速缓存大小计算 C=SxExB。
   - (22)课本家庭作业 6.36（455页）类似题目，分析数组使用缓存时的不命中率。
   - (23)家庭作业 6.34，判断矩阵缓存是否命中。
6. - (22、23)判断符号是全局、局部、外部，是强是弱。
    - (22、23)练习题7.3改编，循环引用的编译命令。
    - (22、23)练习题7.5改编，进行重定位位置计算。
7. - (22)练习题8.3改编，画出进程图。
   - (22)练习题8.4改编，画进程图，写出各种可能的输出。
   - (23)家庭作业8.23改编，练习题8.16改编，加了个循环。
8. - (22)课本图9-20，练习题9.4，题目提供数据，进行虚拟地址翻译、访问缓存。
   - (23)第一问考课本579页的多级页表结构；第二问给虚拟地址从TLB或页表那PPN，然后物理地址去高速缓存判断是否命中，类似课本573页。家庭作业9.12改编。

可以看出，整体题量偏大，考试时间紧张。上述题目出自22与23级学长/学姐，我这一级考试与上一级相差不大，因此下一级可以参考，但我无法保证之后的考试也都是这类型的题目。

如果你希望追求稳定，建议考前1-2周还是都过一遍课本与课本练习题；如果实在没时间，那就利用llm针对上述题目提问，强行overfit记住每一题的解题方法，考试套用。

### 操作系统
考研408科目之一，自然有很多考研资料[^9]。**3学分，均分78.06**。

课程最后还有Linux与Windows的部分，需要记住一些概念，该部分在常规保研辅导书上没有。

常规学习可以向AI问PPT，以及做王道书上的题目。往年卷参考价值没那么大。

这一级的题型如下：
1. 15道单选，一题一分
2. 25道填空，一题一分
3. 6道简答题，每题5分
4. 5道应用题，每题6分

可以看到，前面单选与填空题量较大，可以涉及到许多小的考点。后面大题可以参考2016-2017操作系统期末卷、2022-2023操作系统期末试卷，但题量比往年卷更大。

### 汇编语言与接口技术
**3学分，均分79.17**。期末占比70%，剩下10%是ARM架构论文+大数乘法实现，20%是个人完成汇编游戏/工具，期末前进行答辩。

喜欢我2026年考你远古x86汇编吗？喜欢一学期的csapp与汇编使用完全不同的表示方法吗？

所幸这门课与往年卷重合度很高，重点参考**2015、2017、2019、2020的真题**，我这一级的题目基本往年都考过。

学习前可以先掌握CSAPP中汇编部分，之后再来看这门课。PPT上信息可能较少，不如直接看课本。

题型如下：
1. 单选15题，每题一分
2. 多选5题，每题两分
3. 填空5题，每题两分
4. 简答题6题，每题五分
   1. movsd指令，执行后写出esi，dsi的值
   2. 写出`A=C*D-4`的指令
   3. CS EIP如何得到物理地址
   4. lea和mov offset在寻址、执行、结果的区别
   5. 利用移位和加法实现 `x=x*21`
   6. 用lea实现 `esi=edi*2-18`，并说明实现原理
5. 反汇编C语言题，20分，有四小问，可以参考2015级的反汇编题目
6. 手写汇编大题，15分，具体如下
   - 写出完整的32位汇编代码，输入两个字符串，在大小写不敏感的情况下，比较字符串是否相等。例如AbCD与aBCd是相等的。如果相等，输出Yes；否则输出No。

### 计算机视觉
计拔的必修课，睿信计科与特立计科好像都没有。**3学分，均分81.80**。半开卷考试，也就是可以将知识点写在/打印在一张正反的A4纸上。

虽然BIT101课程评价中，不少同学对fy怨声载道；但我感觉你只要不去听她讲了什么，关系不大。

课程有大作业。在AI发达的今天，不算什么大问题。除了大作业答辩之外，似乎没怎么点过名。课程复习可以看PPT。

### 社会实践
**2学分，均分86.96** 。毛概课程后期会有相关老师通知，大三上开学后提交。没啥好说的，多用AI。

### （选修）计算机新技术专题
**2学分，均分90.7**。挂名的老师是黄河燕，实际每节课都是一位新老师讲自己的研究方向。课程主要是大作业论文，比较方便解决。除此之外，还会有几次点名，有时间最好还是去教室。

### （选修）自然语言处理初步
**2学分，均分89，老师为高扬** 。大作业是小组完成一个研究形式的大模型项目。如果你或同学已经发表过llm相关论文，也可以直接使用。小作业使用线上的平台。

### （选修）强化学习
**2学分，均分88.96，老师为李国政、刘驰**。老师讲的还行，虽然我也没怎么听。大作业是在开悟平台上，训练王者荣耀的游戏Agent。这个环境很恶心，你只能在线上编辑器中编辑，且该环境似乎不与外网相连，不方便使用Coding Agent。可能方案是想办法将代码下载下来，然后本地使用Coding Agent。

不过，最后发现其实也不怎么需要改代码。去年排名较高的组，只是简单地训练了100h，甚至没怎么调参数。所以建议尽量早开始训练。

## 大三下

### 计算机网络

3学分。作为408科目之一，自然有相当多考研资料。不过我没有刷相关考研题目，而是根据ppt+往年卷复习。相关科普书有《网络是怎样连接的》[^10]，可以用几天时间看完。

时间紧张情况下（1-2天时间内），完全可以按照题型问AI进行overfit，掌握每种大题的公式，套用即可。

考试全英文，可以参考2013级试卷。今年题型如下，共10道大题：

1. 20道选择题，共20分。与2013级题目不同，其中大多数题目是概念判断题，少数是计算题。复习时建议看ppt熟悉英文名词与相关概念。
2. 第三章，三种协议，10分。三种协议指Stop-and-Wait、Go-Back-N、Selective Repeat，求它们的利用率，以及sequence number和利用率的关系。
3. 第四章，交换机，10分。题目提供网络的图，让你填写交换机中表的变化。
4. 第五章，OSPF协议，9分。可以参考2013级的第6题。已知邻接表，需要填写一些表，并画出网络拓扑图。还会问OSPF的几种报文类型。
5. 第五章，ARP，8分。已知拓扑图，A向B传递信息，要写出传递信息之前ARP请求、应答帧的内部结构。包括以太帧的Dest与Source MAC，ARP报文内部的Sender/Target各自的MAC/IP。
6. 第五章，划分子网和路由聚合，8分。参考2013级的第7题。本题是升级版，按顺序划分4000、2000、4000、8000个，写出IP地址范围和子网掩码。
7. 第五章，填路由表，7分。已知网络拓扑图，要求填路由表。最后填写tracert命令的回答，也就是写路径。
8. 第六章，PDU分析，8分。参考2013级第10题，相差不大。
9. 第六章，TCP拥塞窗口，10分。今年没有给图，给的是英文描述，要求写出窗口大小，以及握手时SYN、ACK等的值。
10. HTTP传输，10分。要求计算non-persistent、persistent情况下的RTT。

### 计算机组成与体系结构

3学分。依旧408科目之一。

期末题型如下：
1. 10道选择题，每题3分
2. 8道大题，共70分

考试题目基本都源自乐学作业题改编，考前建议复习相应作业题。试卷不多，例如2022～2023年试卷，参考价值没那么大。

我重新看作业题，能回忆起来的一些题型：
1. 第一章，求CPI、MIPS相关。
2. 第二章，二进制的源码、补码、反码转换。
3. 第二章，有3/16、-5/16两个数，要求使用补码计算乘法，写出寄存器的个数与作用，以及详细计算过程。
4. 第二章，给一些指令与出现频率，要求使用Huffman编码，写出编码结果，并计算平均长度。
5. 第四章，给一些存储芯片，求需要多少片，以及相应逻辑结构图。
6. 第四章，页式虚拟存储系统，已知访问页面的顺序，要求写出FIFO、LRU与OPT三种算法的情况表，并计算命中率。
7. 第五章，已知微指令与微命令信号的表，要求安排微指令的操作控制字段格式，并将全部微指令代码化。
8. 第五章，已知流水线处理机的预约表，要求写出延迟禁止表F、冲突向量C，画出流水线状态转移图（还有些别的忘了）。

### 软件工程基础

2学分。有小组项目，需要提交一大堆文档，但是并不需要提交真正的代码或最终结果。

这么课我都没怎么学，基本全靠考前一两天突击。可以参考一些往年卷。

考试题型如下：
1. 20道选择题
2. 简答题1，选择开发模型并提供理由
3. 简答题2，根据描述，确定是哪一种软件维护，并说明理由
4. 大题1，给定情景（员工职级系统），画数据流图，写数据字典
5. 大题2，根据代码，画盒图
6. 大题3，根据代码，自己标号画流图，求环形复杂度，以及独立路径与测试用例
7. 大题4，根据题目描述（指纹开锁系统），画顺序图

### 编译原理与设计
3学分。考试题量较大，主要参考2023-2024学年的试卷（2021级），题型基本固定。今年考得比21级试卷简单。

今年题型如下（卷面总共70分，最后期末占比也是70分）：
1. 10道选择题，共10分，相对简单
2. 10道填空题，共20分，涉及的知识点有：T形图，DFA转换为正则式，由正规式推文法，消除公共前缀转换成LL文法，逆波兰式转C语言，FOLLOW集计算，部分概念
3. 大题1，10分，已知正规式，要求转换成NFA，然后转换成DFA，并且最小化
4. 大题2，10分，已知文法，判断是四种LR文法中的哪一种；该部分计算繁杂，需要很多时间
5. 大题3，10分，已知一段C语言程序，要求写出其中C语言的上下文无关文法描述，以及将代码转换为四元式，结果有20多行；程序涉及if else、while、条件运算、赋值等部分，例如`while(a==b&&(c==d||b==d)){}`
6. 大题4，10分，已知四元式代码段，要求划分基本块、对其中最长的基本块进行DAG、画出控制流图、写出每个节点的必经节点集、找出回边与循环



### 形势与政策Ⅵ
说来搞笑，这学期形策在课程开放期间没点进去过，结果考试期间也进不去。询问老师后，说等通知。几周后通知下来，拉了个新的智慧树网络课去考试。最后60分刚好过。
